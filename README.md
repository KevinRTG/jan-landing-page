## ğŸ” Overview
`jan-nusantara` adalah website company profile dengan sistem katalog produk yang responsif dan aman.  
Fokus utama proyek ini adalah:
- Menampilkan informasi perusahaan secara profesional.

## âœ¨ Features
- **Landing Page**: Company profile CV. JAN NUSANTARA dengan desain modern dan responsif.  
- **Admin Authentication**: Login khusus admin untuk mengakses dashboard.  
- **Dashboard Admin**: CRUD katalog produk konstruksi (tambah, edit, hapus).

## ğŸ› ï¸ Getting Started

### Prerequisites
- Node.js & npm/yarn/pnpm/bun  
- Git untuk clone repository  
- Akun Supabase untuk backend  

### Installation
```bash
git clone https://github.com/KevinRTG/jan-nusantara
cd jan-nusantara
npm install
```

### Usage
Jalankan development server:
```bash
npm run dev
# atau
yarn dev
# atau
pnpm dev
# atau
bun dev
```
Buka [http://localhost:3000](http://localhost:3000) di browser.  
Edit halaman utama di `app/page.tsx`. Perubahan langsung terlihat berkat hot-reload.
